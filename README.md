# Text-to-image-generation-using-stable-diffusion
AI-powered Text-to-Image generation using Stable Diffusion and Hugging Face Diffusers to create high-quality images from natural language prompts.



A Deep Learning project that generates high-quality images from natural language text prompts using the **Stable Diffusion** model from the Hugging Face Diffusers library.

This project demonstrates how pretrained diffusion models can transform textual descriptions into realistic and artistic images with the help of modern generative AI techniques.

---

## 📖 Project Overview

Text-to-Image generation is one of the most popular applications of Generative AI. In this project, a Stable Diffusion model is used to convert user-provided text prompts into visually appealing images.

The project is implemented using the **Diffusers** library and can be executed on **Google Colab** or a local machine with GPU support.

---

## ✨ Features

* Generate images from text prompts

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/text-to-image-generation-using-stable-diffusion.git
```

Move to the project directory

```bash
cd text-to-image-generation-using-stable-diffusion
```

Install the required libraries

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the Jupyter Notebook

```bash
jupyter notebook
```

or

Open **Text_to_image.ipynb** in Google Colab and execute all cells.
